📦 Enable Large File Uploads
Set up Supabase Storage to send videos, large images and documents of any size.
1
Go to your Supabase Dashboard → Storage → Create a new bucket named chat-assets (set to Public)
2
Go to Storage → Policies → Add these two policies for the chat-assets bucket:
INSERT policy (allows upload):
CREATE POLICY "Allow authenticated uploads" ON storage.objects FOR INSERT TO authenticated WITH CHECK (bucket_id = 'chat-assets');
SELECT policy (allows view):
CREATE POLICY "Allow public read" ON storage.objects FOR SELECT USING (bucket_id = 'chat-assets');
3
Run both SQL commands in Supabase → SQL Editor → Refresh Orbit → Files will upload instantly! ✅
📌 This is a one-time setup. Supabase free tier gives you 1GB storage — enough for hundreds of videos.
